## Description

Growing code est mon premier projet de la picine python de 42. Le but est de ce famialiariser et de voir les base du python avec des exercices simples sur le theme du jardin.

## Detail

| Exercices | Role |
|-----------|------|
| ex0 | Le but de cet exercice est d'utiliser simplement la fonction print qui permet d'afficher du texte. |
| ex1 | Le but de cet exercice est de voir la fonction input qui permet de recuperer une information sur le standard et utiliser la fonction print. |
| ex2 | Le but de cet exercice est de faire une multiplication en utilisant une autre maniere d'utiliser la fonction input et la fonction print. |
| ex3 | Le but de cet exercice est de faire une addition de plusieurs input et la fonction print. |
| ex4 | Le but de cet exercice est de voir les conditions if/else et print. |
| ex5 | Le but de cet exercice est de voir les conditions if/else et print. |
| ex6 | Le but de cet exercice est de voir les boucles avec for en iterative, voir la recursive et la fonction range. |
| ex7 | Le but de cet exercice est de voir les conditions if/elif et les strings methods. |

---

## Instructions
Mettre le main.py avec les autre exercices et compiler.
### Complitation
```bash
python3 main.py
```
Par la suite on peu tester un exercice par un exercice ou tout les exercices.
